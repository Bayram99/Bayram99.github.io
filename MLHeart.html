PREDICITING HEART DISEASE USING MACHINE LEARNING

This paper includes 6 phases (objectives) to obtain the best performing model for estimating 10 years of Coronary heart disease.
Phase 1 - Concerns gathering data and importing it into a data frame, this includes importing dataset from csv file into python.
Phase 2 - Involves data visualisation and exploration (EDA) to visualise the relationship between various dataset features and understanding the data.
Phase 3 - Deals with feature engineering, this includes applying data pre-processing techniques such as cleaning records with missing values, standardisation, dimensionality reduction and balancing the dataset.
Phase 4 - Includes model tuning using cross validation technique.
Phase 5 - Incorporates model prediction and evaluation methods to find best performing model.
Phase 6 - Discuss hyperparameter tunning to optimise model performance.
