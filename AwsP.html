Amazon Cloud Computing Application

The architecture, implementation, and assessment of University library application using Amazon Web Services are discussed in this study. The application allows users to upload lectures / labs / assignments that can be accessed from anywhere in the world. The Cloud Application makes use of Amazon Web Services including VPC, Subnets, EC2, S3, CloudFront, RDS, CloudWatch Alarm, ELB, CloudTrail, VPN, IAM and AutoScaling.

Once the application was deployed in the cloud, Testing was conducted to determine whether the Amazon Web Services had been set up correctly and the application is working correctly after the movement from the local XAMPP to the cloud.
This project also emphasises security best practices which include security groups, cryptography and IAM roles. Cost Breakdown using AWS Pricing Calculator is utilised as it provided an estimate on the cost of the whole AWS infrastructure for range of users.
